# intelliDocs Marketing & Conversion Specification

## 1. Goal
**Sole Objective:** To convince enterprise clients, legal firms, and developers to purchase/subscribe to the intelliDocs platform. This is a high-impact, modern marketing landing page designed for conversion.
**Target Domain:** idlp.agentic-sw.com

## 2. Strategic Narrative (The Hook)
*   **Headline:** Stop Searching. Start Knowing.
*   **Sub-headline:** The first enterprise-ready RAG Document Hub that speaks your data's language. Built for scale, integrated with MCP, and ready for your existing workflows.

## 3. Key Selling Points (The "Why Buy")
1.  **"Your Data, Your Truth":** Promote the Filtered Vector Search and RAG capabilities as the end of "hallucinations" in AI.
2.  **"The MCP Advantage":** Position MCP (Model Context Protocol) support as a future-proof feature that lets their existing AI tools (Claude/Gemini) work directly with their secure documents.
3.  **"API-First Architecture":** Convince developers and CTOs with the robustness of the API for custom enterprise integrations.
4.  **"Security by Design":** Highlight the on-prem/cloud options and data sovereignty.

## 4. Page Structure (High-Conversion Flow)
1.  **Hero Section:** High-impact visual (AI document visualization), aggressive CTA ("Get Started Today").
2.  **Feature Grid:** Interactive cards for RAG, Vector Search, and Chat.
3.  **Developer Section:** Highlight MCP and API capabilities with code snippet visuals.
4.  **Pricing/ROI Section:** Clear value-driven tiers focusing on time saved.
5.  **Social Proof:** Placeholder section for testimonials and trusted partner logos.
6.  **Final CTA:** A bottom-of-page "Ready to transform your workflow?" banner.

## 5. Technical Delivery
*   **Framework:** Next.js (App Router) for SEO and performance.
*   **Local Port:** 4777.
*   **Aesthetic:** Dark mode primary with vibrant accents (indigo/violet), high-quality glassmorphism effects, and smooth reveal animations.
